# Bootstrap-Vue Migration Guide

## Decision: bootstrap-vue-next vs PrimeVue vs BootstrapVue3

### Recommended: bootstrap-vue-next
- **Pros**: Closest API to bootstrap-vue, easier migration
- **Cons**: Still in active development, smaller community
- **Package**: `bootstrap-vue-next@^4.0.0`

### Alternative: PrimeVue
- **Pros**: Mature, well-documented, large component library
- **Cons**: Different API, more learning curve
- **Package**: `primevue@^3.0.0` or `primevue@^4.0.0`

## Component Mapping (bootstrap-vue-next)

### Modal Components
```vue
<!-- Vue 2 (bootstrap-vue) -->
<b-modal id="my-modal" :visible="show" @change="show = $event">
  Content
</b-modal>

<!-- Vue 3 (bootstrap-vue-next) -->
<BModal v-model="show" id="my-modal">
  Content
</BModal>
```

**Changes:**
- `:visible` → `v-model`
- `@change` → Use `v-model` with `update:modelValue`
- `$bvModal.show()` → `showModal()` from composable

### Form Components
```vue
<!-- Vue 2 -->
<b-form-input v-model="value" />
<b-form-select v-model="value" :options="options" />
<b-form-checkbox v-model="checked" />
<b-form-radio v-model="selected" :options="options" />
<b-form-textarea v-model="text" />

<!-- Vue 3 -->
<BFormInput v-model="value" />
<BFormSelect v-model="value" :options="options" />
<BFormCheckbox v-model="checked" />
<BFormRadio v-model="selected" :options="options" />
<BFormTextarea v-model="text" />
```

**Changes:**
- Component names: PascalCase (optional but recommended)
- Props mostly remain the same

### Table Components
```vue
<!-- Vue 2 -->
<b-table :items="items" :fields="fields">
  <template #cell(actions)="row">
    <b-button @click="handleClick(row.item)">Action</b-button>
  </template>
</b-table>

<!-- Vue 3 -->
<BTable :items="items" :fields="fields">
  <template #cell(actions)="{ item }">
    <BButton @click="handleClick(item)">Action</BButton>
  </template>
</BTable>
```

**Changes:**
- Slot props: `row.item` → `{ item }` (destructured)

### Navigation Components
```vue
<!-- Vue 2 -->
<b-tabs v-model="tabIndex">
  <b-tab title="Tab 1">Content 1</b-tab>
  <b-tab title="Tab 2">Content 2</b-tab>
</b-tabs>

<!-- Vue 3 -->
<BTabs v-model="tabIndex">
  <BTab title="Tab 1">Content 1</BTab>
  <BTab title="Tab 2">Content 2</BTab>
</BTabs>
```

### Layout Components
```vue
<!-- Vue 2 -->
<b-row>
  <b-col cols="12" md="6">Content</b-col>
</b-row>

<!-- Vue 3 -->
<BRow>
  <BCol cols="12" md="6">Content</BCol>
</BRow>
```

### Button Components
```vue
<!-- Vue 2 -->
<b-button variant="primary" @click="handleClick">Click</b-button>
<b-button-group>
  <b-button>Button 1</b-button>
  <b-button>Button 2</b-button>
</b-button-group>

<!-- Vue 3 -->
<BButton variant="primary" @click="handleClick">Click</BButton>
<BButtonGroup>
  <BButton>Button 1</BButton>
  <BButton>Button 2</BButton>
</BButtonGroup>
```

### Dropdown Components
```vue
<!-- Vue 2 -->
<b-dropdown text="Dropdown">
  <b-dropdown-item @click="action1">Action 1</b-dropdown-item>
  <b-dropdown-item @click="action2">Action 2</b-dropdown-item>
</b-dropdown>

<!-- Vue 3 -->
<BDropdown text="Dropdown">
  <BDropdownItem @click="action1">Action 1</BDropdownItem>
  <BDropdownItem @click="action2">Action 2</BDropdownItem>
</BDropdown>
```

### Alert Components
```vue
<!-- Vue 2 -->
<b-alert :show="showAlert" variant="success">Message</b-alert>

<!-- Vue 3 -->
<BAlert v-model="showAlert" variant="success">Message</BAlert>
```

### Toast/Notification Components
```vue
<!-- Vue 2 -->
this.$bvToast.toast('Message', {
  title: 'Title',
  variant: 'success'
})

<!-- Vue 3 -->
import { useToast } from 'bootstrap-vue-next'
const toast = useToast()
toast.show('Message', {
  title: 'Title',
  variant: 'success'
})
```

## Migration Steps

1. Install bootstrap-vue-next:
```bash
pnpm add bootstrap-vue-next bootstrap@5.3.0
```

2. Update plugin registration:
```javascript
// src/plugins/Bootstrap.js
import { createBootstrap } from 'bootstrap-vue-next'
import 'bootstrap/dist/css/bootstrap.css'
import 'bootstrap-vue-next/dist/bootstrap-vue-next.css'

export default {
  install(app) {
    app.use(createBootstrap())
  }
}
```

3. Update component imports (if using individual imports):
```javascript
// Before
import { ModalPlugin, ButtonPlugin } from 'bootstrap-vue'

// After
import { BModal, BButton } from 'bootstrap-vue-next'
```

4. Update modal show/hide methods:
```javascript
// Before (component-base.vue)
showDialogById(dialog_id) {
  this.$root.$emit('bv::show::modal', dialog_id)
}

// After
import { useModal } from 'bootstrap-vue-next'

showDialogById(dialog_id) {
  const { show } = useModal()
  show(dialog_id)
}
```
