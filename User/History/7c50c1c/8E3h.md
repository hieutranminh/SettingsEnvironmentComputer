# Print Structure Improvements

## Overview

The print system has been refactored to be more efficient, consistent, and maintainable. All dimensions now use pt (points) units consistently, and the header function is reusable across different print pages with customization capabilities.

## Key Improvements

### 1. Consistent Unit System (pt)

- All dimensions now use pt (points) instead of mixing mm and pt
- Conversion utilities provided: `mmToPt()` and `ptToMm()`
- Better precision and consistency across different PDF generators

### 2. Flexible Header System

- Reusable `renderPdfHeader()` function
- Customizable header content per report type
- Support for custom fields with positioning
- Configurable date ranges and subtitles

### 3. Type-Safe Configuration

- Strongly typed interfaces for all print configurations
- Clear separation between header, page, and data configurations
- Better IntelliSense support and error catching

## New Structure

### Types (`src/types/print-preview.types.ts`)

```typescript
// Header customization
interface PrintHeaderConfig {
  title: string
  subtitle?: string
  dateRange?: { from: string; to: string }
  customFields?: Array<{
    label: string
    value: string
    position: 'left' | 'center' | 'right'
  }>
  showCurrentDate?: boolean
  showPageNumber?: boolean
}

// Page configuration (all in pt)
interface PrintPageConfig {
  orientation: 'portrait' | 'landscape'
  paperSize: 'A4' | 'A3'
  margins: { top: number; bottom: number; left: number; right: number }
  font: { family: string; size: number; color: string }
  spacing: { section: number; line: number }
}

// Main configuration
interface PrintPreviewConfig {
  header: PrintHeaderConfig
  page: PrintPageConfig
  exportType: 'pdf' | 'excel' | 'both'
  customData?: Record<string, unknown>
}
```

## Usage Examples

### Basic Usage

```vue
<script setup lang="ts">
import { usePrintPreview } from '@/composables/usePrintPreview'
import { usePrintConfig } from '@/composables/usePrintConfig'
import { PRINT_PREVIEW_WORKER_ACTIONS } from '@/constants/print-preview.constants'

const { openPreview } = usePrintPreview()
const { createPrintConfig } = usePrintConfig()

const handlePrintPreview = () => {
  const config = createPrintConfig('My Report', {
    subtitle: 'Detailed Analysis',
    dateRange: { from: '2025-01-01', to: '2025-01-31' },
    orientation: 'landscape',
    exportType: 'both',
  })

  openPreview({
    type: PRINT_PREVIEW_WORKER_ACTIONS.BRANCH_SALES,
    config,
  })
}
</script>
```

## Migration Guide

### From Old Structure

**Before:**

```typescript
openPreview({
  workerType: PRINT_PREVIEW_WORKER_ACTIONS.BRANCH_SALES,
  headerTitle: 'Branch Sales',
  headerDate: ['2025-01-01', '2025-01-01'],
})
```

**After:**

```typescript
const config = createBranchSalesConfig({
  from: '2025-01-01',
  to: '2025-01-01',
})

openPreview({
  type: PRINT_PREVIEW_WORKER_ACTIONS.BRANCH_SALES,
  config,
})
```
