# 🐛 Phân tích bug: Text render lố ra ngoài phạm vi khi break page

## 🔍 **Nguyên nhân chính**

### ❌ **Vấn đề trong hàm `addNewPage` (line 122-133)**

```typescript
const addNewPage = (renderContext: TextRenderingContext): number => {
  const { pdf, config, context } = renderContext
  pdf.addPage()
  const pageNumber = pdf.getNumberOfPages()

  if (!context.pagesWithHeaders.has(pageNumber)) {
    addHeader(pdf, config, SPACING.MARGIN.TOP)  // ← Header được add với Y mới
    context.pagesWithHeaders.add(pageNumber)
  }

  // ❌ BUG: Return headerEndPosition của PAGE CŨ, không phải page mới!
  return context.headerEndPosition + TEXT_CONSTANTS.SPACING_AFTER_HEADER
}
```

### 🔄 **So sánh với implementation đúng trong usePdfCanvas**

```typescript
// ✅ ĐÚNG - usePdfCanvas (line 157-159)
pdf.addPage()
```

## 📊 **Trình tự lỗi xảy ra**

1. **Page 1**: Header end position = 70 (ví dụ)
2. **Text được render** và đến cuối page 1
3. **Trigger page break**: Gọi `addNewPage()`
4. **Page 2 được tạo**: Header mới được add ở Y = 50 (SPACING.MARGIN.TOP)
5. **Header của page 2 kết thúc** ở Y = ~92 (50 + header height)
6. **❌ BUG**: Return về `context.headerEndPosition + 10` = 70 + 10 = 80
7. **Text bắt đầu render** ở Y = 80 (THẤP HƠN header end của page 2!)
8. **Kết quả**: Text đè lên header hoặc render sai vị trí

## ✅ **Giải pháp**

### Cách 1: Update headerEndPosition cho page mới
```typescript
const addNewPage = (renderContext: TextRenderingContext): number => {
  const { pdf, config, context } = renderContext
  pdf.addPage()
  const pageNumber = pdf.getNumberOfPages()

  let newY = SPACING.MARGIN.TOP
  if (!context.pagesWithHeaders.has(pageNumber)) {
    newY = addHeader(pdf, config, SPACING.MARGIN.TOP)
    context.pagesWithHeaders.add(pageNumber)
    // ✅ Update headerEndPosition for new page
    context.headerEndPosition = newY
  }

  return newY + TEXT_CONSTANTS.SPACING_AFTER_HEADER
}
```

### Cách 2: Return trực tiếp giá trị từ addHeader
```typescript
const addNewPage = (renderContext: TextRenderingContext): number => {
  const { pdf, config, context } = renderContext
  pdf.addPage()
  const pageNumber = pdf.getNumberOfPages()

  if (!context.pagesWithHeaders.has(pageNumber)) {
    const headerEndY = addHeader(pdf, config, SPACING.MARGIN.TOP)
    context.pagesWithHeaders.add(pageNumber)
    return headerEndY + TEXT_CONSTANTS.SPACING_AFTER_HEADER
  }

  // Nếu page đã có header, return margin top + spacing
  return SPACING.MARGIN.TOP + TEXT_CONSTANTS.SPACING_AFTER_HEADER
}
```
