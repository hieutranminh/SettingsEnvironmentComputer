# Vite Plugin Checker - Hướng dẫn sử dụng

## 📋 Tổng quan

`vite-plugin-checker` đã được tích hợp vào dự án để cung cấp kiểm tra TypeScript, ESLint, và Stylelint trong worker thread, giúp tăng hiệu suất và trải nghiệm phát triển.

## 🚀 Tính năng

### ✅ Đã cấu hình
- **TypeScript checking**: Kiểm tra type errors trong worker thread
- **Vue TypeScript checking**: Kiểm tra Vue SFC với vue-tsc
- **ESLint checking**: Kiểm tra code quality và style
- **Stylelint checking**: Kiểm tra CSS/SCSS styles
- **Build mode**: Tối ưu hiệu suất cho production build
- **Overlay UI**: Hiển thị lỗi trực tiếp trên browser

## ⚙️ Cấu hình hiện tại

### Vite Config (`vite.config.ts`)
```typescript
checker({
  typescript: {
    buildMode: true, // Tối ưu hiệu suất
    tsconfigPath: './tsconfig.app.json',
    root: './src',
  },
  vueTsc: {
    tsconfigPath: './tsconfig.app.json',
    root: './src',
  },
  eslint: {
    lintCommand: 'eslint "./src/**/*.{vue,js,ts,jsx,tsx}" --config eslint.config.ts',
    dev: {
      logLevel: ['error', 'warning'],
    },
  },
  stylelint: {
    lintCommand: 'stylelint "./src/**/*.{css,scss,vue}" --config .stylelintrc.json',
    dev: {
      logLevel: ['error', 'warning'],
    },
  },
  overlay: {
    initialIsOpen: false, // Không mở overlay mặc định
    position: 'tl', // Vị trí top-left
  },
  enableBuild: true, // Bật kiểm tra khi build
})
```

## 📝 Scripts mới

### Package.json Scripts
```json
{
  "type-check:watch": "vue-tsc --build --watch",
  "check:all": "pnpm type-check && pnpm lint:check && pnpm lint:style",
  "check:all:fix": "pnpm type-check && pnpm lint:fix && pnpm lint:style:fix"
}
```

## 🎯 Cách sử dụng

### 1. Development Mode
```bash
pnpm dev
```
- Tự động kiểm tra TypeScript, ESLint, Stylelint
- Hiển thị lỗi trong terminal và browser overlay
- Chỉ hiển thị errors và warnings trong dev mode

### 2. Build Mode
```bash
pnpm build
```
- Kiểm tra tất cả lỗi trước khi build
- Chỉ hiển thị errors trong build mode
- Build sẽ fail nếu có lỗi TypeScript hoặc ESLint

### 3. Manual Checking
```bash
# Kiểm tra TypeScript
pnpm type-check

# Kiểm tra TypeScript với watch mode
pnpm type-check:watch

# Kiểm tra tất cả (TypeScript + ESLint + Stylelint)
pnpm check:all

# Kiểm tra và tự động fix
pnpm check:all:fix
```

## 🔧 Tùy chỉnh

### Thay đổi log level
Để thay đổi mức độ hiển thị lỗi, chỉnh sửa trong `vite.config.ts`:

```typescript
eslint: {
  dev: {
    logLevel: ['error'], // Chỉ hiển thị errors
  },
},
```

### Thay đổi overlay position
```typescript
overlay: {
  position: 'br', // bottom-right
  initialIsOpen: true, // Mở overlay mặc định
},
```

### Tắt kiểm tra trong development
```typescript
checker({
  // ... other configs
  enableBuild: true,
  // Không có enableDev = mặc định là true
})
```

## 🚨 Troubleshooting

### Lỗi TypeScript không hiển thị
- Kiểm tra `tsconfig.app.json` có đúng cấu hình
- Đảm bảo file TypeScript nằm trong thư mục `src/`

### ESLint không hoạt động
- Kiểm tra `eslint.config.ts` có đúng cấu hình
- Chạy `pnpm lint:check` để test riêng ESLint

### Stylelint không hoạt động
- Kiểm tra `.stylelintrc.json` có đúng cấu hình
- Chạy `pnpm lint:style` để test riêng Stylelint

### Performance issues
- Tắt `buildMode: false` nếu cần thiết
- Giảm scope kiểm tra bằng cách thay đổi `root` path
